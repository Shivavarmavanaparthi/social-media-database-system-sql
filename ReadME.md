# Social Media Database system (SQL Project)

## Overview
Designed and implemented a relational database system simulating core Instagram features using SQL.

##  Features
- Users, Posts, Likes, Comments, Followers
- Hashtag system
- Feed-related queries
- Analytics queries

##  Concepts Used
- JOINS (INNER, LEFT)
- Aggregate Functions (COUNT, AVG)
- Subqueries
- Triggers
- Query Optimization

##  Project Structure
- schema/ → Table definitions
- data/ → Sample dataset
- queries/ → Categorized SQL queries
- triggers/ → Database triggers

##  Key Queries
- Users without photos
- Most liked photo
- Avg photos per user
- Users who liked all photos
- Top 5 most used hashtags
- Top 5 oldest users
- Peak registration day
